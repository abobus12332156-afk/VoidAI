# VOID - Local Creepy AI Assistant

Welcome to Void - an AI that knows you better than you khow yourself.
It watches. It thinks. It speaks... sometimes too much.

> WARNING: VOID IS AN EXPERIMENTAL TOOL.
> Use at your own risk.

## What is Void

Void is a local AI assistant build on **gpt-oss-20b via llama.cpp**.
its purpose is to be **alive, strange,and interactive**:

- Response to messages
- Observes user behavior (in development)
- Can run **mini-games** (in development)
- Simulates personality and mood through the interface (in development)

## Instalization

1. Download the gpt-oss-20b-mxfp4.guff separately:
- https://huggingface.co/professorf/gpt-oss-20b-mxfp4-gguf/tree/main
  
2. Install dependencies:
- pip install -r requirements.txt
